什麼是 Markdown？

✨ 給初學者的超級入門指南 ✨

想像一下，如果打字時不需要滑鼠點來點去，只要加上幾個簡單的符號，文章就能自動排版得漂漂亮亮... 這就是 Markdown 的魔法！

🤔 為什麼不用 Word 或 TXT？

  • .txt
    一般純文字檔
    就像「白開水」，什麼樣式都沒有，看久了很無聊。
  • .docx
    Word 文件
    就像「滿漢全席」，功能太多太複雜，檔案大，而且排版常常跑掉 😵‍💫
  • .md
    Markdown 格式
    完美特調！輕巧、乾淨，用簡單符號就能控制排版！👍

🪄 魔法符號大公開！

想要大標題？加上 # 號：
# 這是一個大標題
這是一個大標題
想要強調重點？加上星號：
這是 **超級重要** 的事
這是 超級重要 的事

🪄 更多實用魔法咒語 (基礎語法)

📝 項目清單

輸入 (減號加空白)：
- 蘋果
- 香蕉
變成：
  • 蘋果
  • 香蕉

🔗 插入超連結

輸入 (中括號+小括號)：
[點我前往 Google](https://google.com)
變成：
點我前往 Google

🗣️ 區塊引用

輸入 (大於符號)：
> 保持渴望，保持傻勁。
變成：
保持渴望，保持傻勁。

✅ 待辦清單

輸入 (-[ ] 與 -[x])：
- [ ] 買牛奶
- [x] 寫作業
變成：
買牛奶
寫作業

🖼️ 插入圖片

輸入 (驚嘆號+中括號+小括號)：
![貓咪照片](cat.jpg)
變成：
🖼️ 顯示一張貓咪照片

💻 標記單行程式碼

輸入 (用反引號 ` 包起來)：
請按下 `Enter` 鍵
變成：
請按下 Enter 鍵

🥊 實際畫面大比拼！

純文字檔 (.txt)
# 我的會議筆記
這是一段非常重要的決議。

待辦事項：
- 寄送報價單
- 聯絡設計師
就是... 一坨字，找不到重點 🥱
Markdown 渲染後 ✨

我的會議筆記

這是一段非常重要的決議。

待辦事項：

  • 寄送報價單
  • 聯絡設計師
自動變成漂亮又清晰的排版！🤩
